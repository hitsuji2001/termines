### Description

A Minesweeper game that can be interactively played on terminal

### Usage

``` bash
$ ./build.sh
$ ./termines
```

### Control

| Key              | Function         |
|------------------+------------------|
| <kbd>w</kbd>     | Move up          |
| <kbd>s</kbd>     | Move down        |
| <kbd>a</kbd>     | Move left        |
| <kbd>d</kbd>     | Move right       |
| <kbd>space</kbd> | Open cell        |
| <kbd>f</kbd>     | Flag/Unflag cell |
| <kbd>r</kbd>     | Restart game     |
| <kbd>Q</kbd>     | Quit             |
